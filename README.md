# DiscordBot

## Installation

```console
$ git clone git@github.com:Pixeloceax/DiscordBot.git
```

```console
$ cd DiscordBot/$BOT_NAME
```

```console
$ npm install
```

## Usage

### Setup

setup your bot token in the **_.env_** file at the root of the project:

```
DISCORD_TOKEN= <your token>
```

### Run

```console
$ npm start
```
